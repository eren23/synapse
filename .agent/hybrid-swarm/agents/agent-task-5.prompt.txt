# Task: Zig Tensor Core (Tensor struct, Views, Iterator)

Implement zig/src/tensor/tensor.zig: Tensor struct wrapping Storage with shape/strides/offset. Typed element access (at/set), numel, is_contiguous check. Implement zig/src/tensor/view.zig: Zero-copy reshape (validates numel match), transpose (swaps strides/shape), slice (adjusts offset+shape+strides). Implement zig/src/tensor/iterator.zig: Strided multi-dimensional iterator for both contiguous (pointer walk) and non-contiguous (index computation).

Tests in zig/tests/test_tensor.zig (extended): Create tensor, read/write elements. Reshape preserves data. Transpose produces correct strides. Slice produces correct sub-view. Iterator visits all elements in correct order. Benchmark: Iterator throughput contiguous vs strided on 1M elements.

Pass/fail: All index operations correct. Contiguous iterator throughput within 10% of raw pointer walk. Reshape/transpose/slice do not copy data (same storage pointer).

Target files: synapse/zig/src/tensor/tensor.zig, synapse/zig/src/tensor/view.zig, synapse/zig/src/tensor/iterator.zig
