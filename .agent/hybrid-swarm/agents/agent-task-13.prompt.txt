# Task: Rust Autograd Ops (All Forward+Backward Implementations)

Implement crates/synapse-autograd/src/ops/: mod.rs, arithmetic.rs (Add/Sub/Mul/Div forward+backward with broadcasting gradient reduction), matmul.rs (dA = dOut @ B^T, dB = A^T @ dOut), reduce.rs (Sum backward expand, Mean backward), activation.rs (ReLU/Sigmoid/Tanh/GELU backward), reshape.rs (Reshape/Transpose backward), conv.rs (Conv2d backward: im2col grad, weight grad, input grad via col2im), pool.rs (MaxPool backward scatter via saved indices, AvgPool backward), batchnorm.rs (input/gamma/beta gradients), softmax.rs (Softmax/LogSoftmax backward), loss.rs (MSELoss/CrossEntropyLoss).

Tests: For EVERY op: grad_check (numerical vs analytical, non-negotiable). MatMul gradient shapes correct for non-square. Broadcasting gradient reduction correct. Benchmark: Forward+backward on 10-layer MLP, batch 64.

Pass/fail: grad_check passes for ALL ops (relative error < 1e-3). No gradient shape mismatches. Forward+backward overhead <=5% vs forward-only.

Target files: synapse/crates/synapse-autograd/src/ops/mod.rs, synapse/crates/synapse-autograd/src/ops/arithmetic.rs, synapse/crates/synapse-autograd/src/ops/matmul.rs, synapse/crates/synapse-autograd/src/ops/reduce.rs, synapse/crates/synapse-autograd/src/ops/activation.rs, synapse/crates/synapse-autograd/src/ops/reshape.rs, synapse/crates/synapse-autograd/src/ops/conv.rs, synapse/crates/synapse-autograd/src/ops/pool.rs, synapse/crates/synapse-autograd/src/ops/batchnorm.rs, synapse/crates/synapse-autograd/src/ops/softmax.rs, synapse/crates/synapse-autograd/src/ops/loss.rs, synapse/tests/integration/autograd_correctness.rs
