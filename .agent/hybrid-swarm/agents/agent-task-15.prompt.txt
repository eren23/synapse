# Task: Rust Data Pipeline (Dataset, DataLoader, Samplers, Transforms)

Implement crates/synapse-data/: dataset.rs (Dataset trait, InMemoryDataset, TensorDataset wrapping features/labels pair), sampler.rs (SequentialSampler, RandomSampler with seed, WeightedRandomSampler), collate.rs (stack tensors along new batch dimension, pad sequences to max length), dataloader.rs (configurable batch_size/shuffle/drop_last, multi-threaded prefetching with double-buffering), transform.rs (Normalize(mean,std), RandomHorizontalFlip, ToTensor).

Tests: DataLoader iterates correct number of batches. Shuffle produces different order each epoch but all elements exactly once. Batch tensors have correct shape [batch,...]. Transforms modify data correctly. Benchmark: DataLoader throughput (batches/sec) with 4 prefetch threads.

Pass/fail: Correct batch count: ceil(N/batch_size) or floor if drop_last. Shuffle covers all elements exactly once per epoch. Prefetch: next batch ready before previous finishes.

Target files: synapse/crates/synapse-data/Cargo.toml, synapse/crates/synapse-data/src/lib.rs, synapse/crates/synapse-data/src/dataset.rs, synapse/crates/synapse-data/src/sampler.rs, synapse/crates/synapse-data/src/collate.rs, synapse/crates/synapse-data/src/dataloader.rs, synapse/crates/synapse-data/src/transform.rs
