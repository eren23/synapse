# Task: Zig Tiled SGEMM (Matrix Multiplication)

Implement zig/src/ops/matmul.zig: Full SGEMM with 8x8 micro-kernel using NEON FMA (or 8x8 AVX2 on x86), L1 tiling (MR=8, NR=8), L2 tiling (MC=256, KC=512), L3 tiling (NC=4096), packing of A and B for cache locality, support for trans_a/trans_b, edge cleanup for non-tile-multiple dimensions. MUST also include naive triple-loop implementation for benchmark comparison.

Tests in zig/tests/test_matmul.zig: Correctness vs naive triple-loop for sizes: 1x1, 8x8, 16x16, 64x64, 128x128, 512x512, 1024x1024, non-square (32x64 * 64x48). Transposed variants. Tolerance 1e-4 for large sizes.

Benchmarks in zig/tests/bench_matmul.zig: 512x512 SGEMM, 100 iterations. Naive vs tiled.

Pass/fail: Correctness within 1e-4 relative error for all sizes. >=5x speedup vs naive triple-loop on 512x512. >=2 GFLOPS on Apple M-series.

Target files: synapse/zig/src/ops/matmul.zig, synapse/zig/tests/test_matmul.zig, synapse/zig/tests/bench_matmul.zig
