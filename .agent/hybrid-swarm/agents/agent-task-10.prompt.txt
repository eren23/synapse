# Task: Zig FFI Exports + C ABI Build

Implement zig/src/ffi/exports.zig: All export fn declarations with C calling convention. Wraps every Zig function with error-code returns (syn_status_t), opaque pointer handles (syn_storage_t, syn_tensor_t, syn_arena_t, syn_pool_t), and null-safety checks. No panics escape. Generate synapse.h C header. Update zig/build.zig to produce libsynapse_zig.a static library with cross-compilation support for aarch64 and x86_64.

Full FFI surface: storage create/retain/release/data/len, tensor create/destroy/shape/ndim/data_ptr/is_contiguous/contiguous, arena create/reset/destroy, pool create/destroy, sgemm, elementwise add/sub/mul/div, relu/sigmoid/tanh/gelu, reduce sum/max/mean, softmax, batchnorm, conv2d, maxpool2d, avgpool2d, transpose, raw SIMD vector ops (vadd/vmul/vfma/vreduce_sum/vreduce_max).

Tests: Round-trip test in Zig via exported C ABI functions. Verify invalid inputs produce proper error codes, not panics.

Pass/fail: All FFI functions compile to valid C ABI. No panics escape across FFI boundary. zig build produces libsynapse_zig.a.

Target files: synapse/zig/src/ffi/exports.zig, synapse/synapse.h
