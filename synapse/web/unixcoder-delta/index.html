CodeDeltaTok · WASM demo

Loads microsoft/unixcoder-base + a trained CodeDeltaTok head into the browser as pure-Rust WASM, then computes the 768-dim delta token for a (before, after) code pair on-device. No server-side inference.

Note: UniXcoder is 480 MB in fp32; point the “UniXcoder safetensors” picker at the file shipped by huggingface-cli download microsoft/unixcoder-base. The CDT head is ~300 MB — export with python scripts/export_unixcoder_reference.py convert-cdt ... before loading.
