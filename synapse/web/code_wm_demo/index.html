v2 vs v1 — what changed

  • Attention pooling replaces CLS token — learnable readout over all patches
  • 3-loop encoder (was 6) — signal lives in early loops, deep supervision preserves it
  • VICReg at readout — prevents representational collapse (eff. rank 53 vs 1.95)
  • 700 vocab (was 662) — expanded AST token set
  • 15-d actions (was 7) — richer edit descriptor for transition prediction
  • Config: encoder_loops=3 aux_loops=1,2 reg_mode=vicreg ema_decay=0.99999
